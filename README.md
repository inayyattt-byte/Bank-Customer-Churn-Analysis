# Bank Customer Churn Analysis

## Project Overview
This project analyzes bank customer data to identify trends and factors contributing to customer churn.

The analysis focuses on customer demographics, account balances, activity status, and geographic differences to better understand customer retention behavior.

---

## Tools Used
- Python
- Pandas
- Matplotlib
- Excel
- VS Code

---

## Dataset
The dataset contains 5,000 bank customer records with information including:
- Credit Score
- Geography
- Gender
- Age
- Balance
- Activity Status
- Estimated Salary
- Customer Churn

Dataset source: Kaggle

---

## Business Questions Analyzed
1. Which country has the highest churn?
2. Do inactive members leave more?
3. Which age group exits most?
4. Do higher balances affect churn?
5. Which gender leaves more?

---

## Key Findings
- Germany showed the highest customer churn rate.
- Female customers had slightly higher churn compared to male customers.
- Older age groups showed marginally higher churn rates.
- Germany also had the highest average account balances.
